🤖 Smart AI — Setup & Guide

Smart AI provides on-screen assistance and quality-of-life tools. Use it responsibly and always comply with the terms and policies of the platforms you use.

Requirements

  • Windows 10/11 (latest updates recommended)
  • Microsoft Visual C++ Runtimes (see links in Troubleshooting)
  • Administrator permissions for initial setup

Installation & First Launch

  1. Download the latest Smart AI installer from your official account area.
  2. Run the installer and follow the on-screen steps.
  3. Start the app, sign in with your account, and complete the onboarding.
  4. Open the overlay settings and adjust visibility, hotkeys, and notifications.

Basic Usage

  • Overlay: Toggle the heads-up panel via your assigned hotkey.
  • Profiles: Create and switch profiles for different games or workflows.
  • Logs: Use the log panel to review recent actions and errors.

Good Practices

  • Keep your OS and drivers up to date.
  • Close conflicting overlays (recorders, screen filters) if you see visual glitches.
  • Back up your profile folder periodically.

Note: Feature availability may vary by region and version. Always follow applicable terms and policies.

🧰 Smart AI — Troubleshooting

If you encounter issues, try the steps below before contacting support.

Common Fixes

  • Restart: Reboot your PC and try again.
  • Run as Admin: Right-click Smart AI → Run as administrator.
  • Update Runtimes: Install the latest Microsoft Visual C++ packages.
  • Disable Conflicts: Temporarily close other overlays/recorders.

Useful Links

  • Microsoft Visual C++ 2015–2019 Redistributable (x64)
  • Latest supported Visual C++ downloads

Collecting Logs

  1. Open Smart AI → Settings → Logs.
  2. Click Export to save the latest session logs.
  3. Attach the log file when contacting support.

If the problem persists, contact support with a short description, screenshots, and the exported logs.
